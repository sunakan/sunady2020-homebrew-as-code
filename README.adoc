= homebrew as code(macenv as code?)

== 以下をコードで管理したい

* HomeBrewで入れるパッケージ群
* Macに入れるGUIアプリ

== どんな時に使える？

* Macを買い替えた時
* Macをクリーンインストールした時
* MacのアプリをポチポチDLして入れたくない時

== Brewfileにない、brewで管理しているパッケージを削除

.確認するだけ
----
$ brew bundle cleanup
----

.実行!!
----
$ brew bundle cleanup --force
----
